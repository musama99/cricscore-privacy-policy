# Usama Apps Studio Website

The official website for **Usama Apps Studio**, a growing portfolio of mobile applications.

The website currently features **CricScore**, an offline cricket scoring app, and is structured to support more apps in the future.

## What this repository includes

- Responsive studio website
- Apps portfolio page
- CricScore product page
- CricScore Privacy Policy
- Support, About, Profile, and Contact pages
- GitHub Pages deployment support
- No build process or external framework required

## Technologies

- HTML
- CSS
- JavaScript

## Get the project

Clone the repository:

```bash
git clone https://github.com/musama99/usama-apps-studio.git
```

Open the project directory:

```bash
cd usama-apps
```

Start a local development server:

```bash
python -m http.server 8000
```

Open the website in your browser:

```text
http://localhost:8000
```

You can also use the **Live Server** extension in Visual Studio Code.

## Deployment

This website can be deployed directly using GitHub Pages.

1. Open the repository settings.
2. Go to **Pages**.
3. Select the `main` branch.
4. Select the root directory.
5. Save the settings.

No build command is required.
